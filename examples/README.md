# mflash Examples

This directory contains example mflash decks and progress files.

## Valid examples

- `minimal-v2.mflash.json` — smallest useful mflash v2 deck.
- `anonymous-share-v2.mflash.json` — shareable v2 deck with no timestamps.
- `progress/sample.progress.json` — sample separate study progress file.

## Invalid examples

The `invalid/` directory is reserved for files that should fail validation. These are useful for testing validators and app loaders.
