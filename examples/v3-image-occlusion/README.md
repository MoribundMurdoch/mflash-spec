# mflash v3 Image Occlusion Example

This example demonstrates image occlusion.

It covers:

- `kind: "image_occlusion"`
- `occlusion.image`
- `occlusion.masks`
- rectangular masks
- ellipse masks
- polygon masks
- normalized coordinates from `0.0` to `1.0`
- image assets under `assets/cards/<card_id>/`
