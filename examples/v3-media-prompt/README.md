# mflash v3 Media Prompt Example

This example demonstrates cards where media is the main prompt.

It covers:

- `kind: "media_prompt"`
- `prompt_image`
- `prompt_animation`
- image and gif media types
- answer fields
- card assets under `assets/cards/<card_id>/`
